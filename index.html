Settings

Saved only in this browser using localStorage. For GitHub Pages, restrict your Google key to https://YOUR_USERNAME.github.io/* and/or https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/*.

Troubleshooting
  1. Make sure index.html, styles.css, and app.js are in the root of the GitHub repo.
  2. Enable both Maps JavaScript API and Places API.
  3. Make sure billing is enabled in Google Cloud.
  4. Add your GitHub Pages URL to allowed HTTP referrers.
  5. Try the fallback Google Maps link if Places fails.
